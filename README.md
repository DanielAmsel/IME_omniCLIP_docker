# omniCLIP_docker
Docker file for creating omniCLIP Docker Container [omniCLIP-Repository](https://github.com/philippdre/omniCLIP)

## Usage
```
docker pull imetools/omniclip_docker
docker run -v DATADIRECTORY:/data imetools/omniclip_docker
```
